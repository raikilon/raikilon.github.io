Team Members:
  • Noli Manzoni (noli.manzoni@students.bfh.ch)
  • Samuel Gauthier (samuelluc.gauthier@students.bfh.ch)
Features:
  • Random maze generation
  • Three game modes (first person, third person and third person isometric)
  • Customized water
  • Personalized menus
Intructions:
  • Use the menu to choose the game mode and the settings
  • Use "1" to return to the main menu when playing
  • Use A,W,D,S to move
  • In third person mode use Q and E to rotate the camera of 90 degree
  • In first person use SHIFT to run
  • Use SPACE to jump
